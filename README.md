# karma-loud

Karma plugin for [Loud][], web accessibility unit-testing framework

## Installing

```
npm install karma-loud --save-dev
```

## Getting Started

```javascript
/* karma.conf.js */
module.exports = function(config) {
    config.set({
        frameworks: ['loud'],
        …
    });
};
```
